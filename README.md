# JalRakshak - AI Waterborne Disease Prediction System

Real-time waterborne disease risk prediction and prevention platform for public health management.

## Features
- AI-powered disease risk prediction
- Interactive risk mapping
- Real-time monitoring dashboard
- Multi-level alert system
- Advanced analytics

## Installation
1. Clone the repository
2. Install dependencies from requirements.txt
3. Run the application with: streamlit run jalrakshak_app.py

## Requirements
- Python 3.8 or higher
- Streamlit 1.28.0
- Pandas 2.1.1
- NumPy 1.24.3
- Plotly 5.17.0

## Project Structure
jalrakshak_app.py - Main application
requirements.txt - Dependencies
synthetic_waterborne_data.csv - Sample dataset
README.md - Documentation

## Quick Start
1. Install Python and required packages
2. Run: streamlit run jalrakshak_app.py
3. Open browser at http://localhost:8501
